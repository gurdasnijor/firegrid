## Current State

The repo uses Firegrid product and package vocabulary:
`@firegrid/protocol` (browser-safe shared schemas), `@firegrid/client`
(browser/app-facing launch client), `@firegrid/runtime` (server-side durable
workflow and launch runtime), and `apps/flamecast` (clean-room tracer app).

Run the canonical ready-for-review gate when preparing a PR:

```sh
pnpm verify
```

Implementation work starts from the Acai specs and includes full
Acai ACID references in test names or nearby comments.

## Architecture Evidence

Dependency graph evidence is generated by dependency-cruiser and committed as
Mermaid files under `docs/`. Refresh it with:

```sh
pnpm run arch:deps
```

Module-level import graphs are available when a PR needs a closer import review:

```sh
pnpm run arch:deps:detail
```

The strict CI boundary gate is:

```sh
pnpm run lint:deps
```

## Run The Runtime

Run the Firegrid runtime against an existing Durable Streams endpoint by setting
`DURABLE_STREAMS_URL`:

```sh
DURABLE_STREAMS_URL=https://... pnpm --filter @firegrid/runtime run firegrid
```
